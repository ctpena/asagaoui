メガメニューは主にグロバールメニューで使用されるドロップダウンメニューです。自由なコンテンツ設計が可能な構造にしました。

## コード

### CSS

```scss
.mega-menu-area {}

.mega-menu {}

.mega-menu-header {}

.mega-menu-content {}

.mega-menu-footer {}
```

### HTML

```html
<div class="mega-menu">
    <div class="mega-menu-header">
    </div>
    <div class="mega-menu-content">
    </div>
    <div class="mega-menu-footer">
    </div>
</div>
```
